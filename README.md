# Cofrinho em Java

Projeto em **Java** que simula um cofrinho de moedas via console.  
Permite adicionar, remover, listar moedas e calcular o valor total convertido para Real.

---

## Objetivo
Praticar **Programação Orientada a Objetos (POO)** utilizando:
- Herança
- Polimorfismo
- Classes abstratas
- Coleções (`ArrayList`)

---

## Estrutura Básica
src/
└─ empresa/
├─ Main.java
├─ Cofrinho.java
├─ Moeda.java
├─ Real.java
├─ Dolar.java
└─ Euro.java

---

## Funcionalidades

Adicionar moedas

Remover moedas

Listar moedas

Calcular total em Real
